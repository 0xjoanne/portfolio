Concept

Multiplayer Web Game

FISHSCAPE! is a real-time multiplayer web action game that offers two ways to enjoy! Each player will be entered into an ocean scenario where one randomly selected player will be created as the villainous shark whose main objective is to rid his kingdom of pesky annoying fish while at the same time feeding his hunger! The remaining players will be created as fish, in which their main objective is to escape the menacing teeth of the shark, while outlasting the time limit to be crowned the fittest, smartest, and last surviving fish of his school!

Design by Erick Hernandez

Development
Process

Production Logic

- Create basic idea of game to get started
- Research best, easiest game library
- Incorporate game library with socket.io
- Design interface and find sprites to incorporate into game
- Create simple game logic
- Create process for beginning and ending a game

Data Flow

Socket.io

From Client: Player info( socket.id, username), Player movements, Phaser.js functionality, Players status(creation and death)

Server: Sort and store players, Gameplay functionality/logic(stages, timer etc.)

Server to other client: Other players movements and death

Challenges

Playable + Enjoyable

- Incorporate Phaser.js with Socket.io
- Creating separate game rooms to play multiple games at once
- Creating an intriguing game logic
