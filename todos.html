Summary

Web App

Todos is a single page web application with Node.js, Mongodb and Pug.

There are main challenges to develop this app:

- Front end development with Pug
- Add, delete, complete todos
- Order todos by creation time
- Save data with MongoDB

Experience

What I got

That was my first time to use MongoDB and Pug. I thought it would be hard to handle it, but I totally changed my mind after this project. It really doesn’t matter what programming language or technology we use. When we get the basic knowledge and have a great understanding the logic of the code, all problem will be solved.
