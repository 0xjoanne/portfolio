Summary

IoT & Node.js

Parking in the city is hard. It happens a lot that there is no parking space left when you arrive at a parking lot. Espacio is a web application which helps users to find Find nearby parkings that have empty space. It grabs parking lot information by using IoT device which is made by Arduino and two Ultrasonic sensors. Espacio tells users where the closest places to park are located. It wants to make city living just a little more convenient. Save money, time and gas.

Backend and IoT setup by Rene Ramirez

Features

What it has

- Get user location by Geolocation or Google Place
- Find nearyby parking lot
- Present parking space left in real-time
- Present basic information and traffic flow of certain parking lot
- Get direction to specific parking lot

Development Choice

Front-end developer

Before starting coding Espacio, there was a hard time to decide which framework to use. Pure javascript or Vue.js?

“Learning by using” is one of my biggest takeaways from the past year. That is to say that developing a project is the best way to learn a new programming language or improve our coding skill. At that time, I just started to learn Vue.js, so I knew it was a great opportunity.

During development, I used Google, Github and Stack Overflow when there were some problems showing up. When one after another features were implemented, I had a better understanding of Vue.js. However, there are so many resources online. Sometimes, I spent a lot of time to find out the solution for one specific problem. I didn’t know which one is best. Those made me frustrated, but I always remember that all roads to Rome.
